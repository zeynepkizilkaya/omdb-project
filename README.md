# movieys

movieys is a pastel themed movie search web application built with HTML, CSS and JavaScript using the OMDb API. My inspiration was the "Whimsy Core" while designing the site.

## Features

- Search movies by title
- View movie details
- Responsive design
- Error handling
- LocalStorage support

## Technologies

- HTML
- CSS
- JavaScript
- OMDb API
